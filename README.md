🧠 Orlena AI:
A real-time multilingual communication system that works across 11 languages
Achieves ~2 second latency, basically matching API response speeds → feels near real-time
Designed to act as a live meeting/communication bridge, removing language barriers
Focused on practical usability (not just translation, but seamless interaction)

⚙️ What it actually does:
Takes speech/text in one language
Processes + translates instantly
Outputs it in another language in real time
Enables cross-language conversations without delay

💡 Why it’s cool:
Solves a real-world problem (language barriers in meetings)
Combines LLMs + speech/translation systems
